## Описание проекта
Данный проект представляет собой учебное веб-приложение, разработанное для демонстрации навыков работы с системой контроля версий Git и верстки веб-страниц.

## Технологии
* Git / GitHub

## Структура проекта
* `index.html` — главная страница приложения.
* `IMAEVA.md` — файл описания проекта (документация).

## Запуск
Открыть файл `index.html` в любом современном браузере.
