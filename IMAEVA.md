## Описание проекта
Данный проект представляет собой десктопное приложение, разработанное в среде Visual Studio для демонстрации навыков работы с системой контроля версий Git и объектно-ориентированного программирования.

## Технологии
* C# / .NET
* Visual Studio 2022
* Git / GitHub

## Структура проекта
* MainWindow.xaml — интерфейс главного окна приложения.
* MainWindow.xaml.cs — логика обработки событий главного окна.
* IMAEVA.md — файл описания проекта (документация).

## Запуск
1. Открыть файл решения (.sln) в Visual Studio.
2. Нажать "Пуск" (F5) для компиляции и запуска приложения.
